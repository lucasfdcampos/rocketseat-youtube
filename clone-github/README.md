## :page_with_curl: Como obter o layout

```bash
  # clone o repositório
  $ git clone https://github.com/lucasfdcampos/rocketseat-youtube.git

  # acesse o diretório
  $ cd clone-github

  # instale as dependências
  $ yarn 

  # inicie o projeto
  $ yarn start
```
